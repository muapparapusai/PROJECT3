What is an HTML Attribute?

  • HTML Attribute provide Additional information About HTML Attribute.
  • Attribute Usually come in name value pairs like :name="value".


The href Attribute:

  • < a> tag define A hyperlink. the href Attribute specifies the URL(Uniform Resource Locators) of the page the link goes to. < /a>


The src Attribute:

  • The < img> tag is used to embed an image in an HTML page.
  • The src Attribute specifies the path to the image to be display.

There are two URL in the src Attribute:

  1. Absolute URL
  2. Relative URL

1.Absolute URL:

  • Links to an external Images that is hosted an another website.

2.Relative URL:

  • Links to an images that is hosted with in the website, the URL does not includes the domain name.

Width and Height Attribute

The < img> tag should also contain the Width and Height Attributes,Whivh specifies the width and height of the image (in pixels).
